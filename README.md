# PyBank

A menu-driven banking system built using Python.

## Features

- Create New Account
- View Account Details
- Deposit Money
- Withdraw Money
- Transfer Money
- Search Account
- View All Accounts
- Delete Account
- Automatic Data Saving
- Automatic File Creation 

## Concepts Used

- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling
- Lists
- Functions
- Loops
- Random Module
- OS Module
